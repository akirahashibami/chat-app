# さんかくひろば（リアルタイムチャットアプリ）

URL
http://18.181.34.125/

## 概要

ActionCableを用いたチャットアプリです。
他のユーザーと色々なコミュニケーションを楽しむことができます。
※railsのバージョンが5と6では書き方が変わるようです  
※今回5で行っています

## 使用技術
- Ruby 2.5.7
- Ruby on Rails 5.2.4
- MySQL 5.7.22
- Nginx
- Puma
- AWS
  - EC2
  - RDS

開発環境
- Virtualbox
- Vagrant
